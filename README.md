# PROTEYE: EGNN-like Protein Structure Reconstruction

> *Статус проекта:* Данный репозиторий является практическим домашним заданием (учебным проектом) в рамках обучения в **Samsung Innovation Campus**. Проект находится в стадии активной разработки (Work in Progress).

## Текущие достижения
В рамках архитектуры **PROTEYE** реализована эквивариантная графовая нейросеть (EGNN-like denoiser) для восстановления зашумленных 3D-координат белков. Удалось интегрировать физические ограничения (**Physics-Informed Loss**):
* **Huber Loss** для стабильной обработки пространственных координат.
* **Bond Length Loss** для удержания идеальной длины связей $C_\alpha - C_\alpha$ (~3.8 Å).
* **Angle Loss** для контроля валентных углов полипептидной цепи на петлях.

### Результаты валидации:
* **1UBQ (Убиквитин):** Итоговый RMSD **0.1980 Å**.
* **1AAR (Независимый тест):** Итоговый RMSD **0.1759 Å**.

* **Оригинальная структура.** Графовая структура белка 1AAR (α1A-адренорецептор)
* **"Шумная" структура.** К идеальным координатам атомов $C_\alpha$ был добавлен случайный гауссовский шум $\sigma = 1.0 \text{Å}$, искусственно деформировавший структуру белка.
* **Реконструированная структура.** Эквивариантная нейросеть (EGNN-like denoiser) сместила атомы из искаженных позиций обратно в их нативные биологические координаты, полностью «счистив» наложенный шум.

* **Распределение локальных ошибок (Per-Residue Error).** График среднеквадратичного отклонения по каждому аминокислотному остатку. Благодаря интеграции `Angle Loss` и `Huber Loss` профиль ошибки стабилизирован на уровне 0.1–0.3 Å для 90% цепи, а локальные пики на подвижных петлях удержаны в рамках физически допустимых пределов (~1.0 Å).
