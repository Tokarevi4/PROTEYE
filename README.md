**PROTEYE** — это научно-исследовательский фреймворк на базе эквивариантных графовых нейронных сетей (EGNN), предназначенный для высокоскоростного денойзинга 3D-координат макромолекул и моделирования альтернативных конформаций белков. 

Проект разработан в рамках финального исследования для **Samsung Innovation Campus**.

---

## 🛠️ Архитектура и Научная новизна

В основе PROTEYE лежит сквозной (End-to-End) конвейер, который преобразует сырые структуры Protein Data Bank (PDB) в пространственные графы контактов на основе радиуса отсечки (**Radius-based Cutoff, R = 10.0 Å**). Каждая вершина графа кодируется 21-мерным One-Hot вектором аминокислотного состава, объединённым с геометрическими инвариантами степеней узлов.

### Physics-Informed Loss Function
В отличие от классических геометрических автоэнкодеров, PROTEYE оптимизируется с учётом фундаментальных физико-химических законов структурной биологии:
1. **Huber Loss ($\mathcal{L}_{\text{coords}}$):** Обеспечивает робастную стабилизацию пространственных координат и защиту от взрыва градиентов на атомах $C_\alpha$.
2. **Bond Length Loss ($\mathcal{L}_{\text{bond}}$):** Жестко штрафует модель за растяжение или сжатие полипептидной цепи, удерживая идеальную длину пептидных связей $C_\alpha - C_\alpha$ вблизи **$~3.8 \text{ Å}$**.
3. **Angle Loss ($\mathcal{L}_{\text{angle}}$):** Контролирует псевдовалентные углы, стабилизируя укладку на вторичных структурах и подвижных петлях.

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{coords}} + \alpha \cdot \mathcal{L}_{\text{bond}} + \beta \cdot \mathcal{L}_{\text{angle}}$$

## Результаты валидации (100 эпох)

Модель обучена на репрезентативной выборке из **4 759 уникальных белковых структур PDB** с использованием динамического сэмплирования шума ($\sigma \in [0.1; 1.5] \text{ Å}$). Ограничение Data Leakage реализовано на уровне индивидуальных белков (Protein-level Split).

* **1UBQ (Убиквитин):** Итоговый геометрический RMSD после денойзинга — **`0.3890 Å`**
* **1AAR ($\alpha_{1A}$-адренорецептор):** Итоговый геометрический RMSD — **`0.4083 Å`**
* **Стабильность топологии:** Средняя ошибка длин связей пептидного остова снижена до **`0.3999 Å`**.



### Визуализация реконструкции (на примере белка 1AAR)
Сеть успешно восстанавливает нативную конформацию, увеличивая биологическую метрику структурного сходства **TM-score до 0.9922** (структура практически идентична нативной).

## Структура репозитория

```text
PROTEYE/
├── data/           # Пайплайн данных (raw_pdb, processed, train)
├── inference/      # Модуль предсказаний и класс ProtEyePredictor
├── models/         # Архитектура Эквивариантной сети (ProtEyeEGNN)
├── scripts/        # Скрипты препроцессинга и статистического анализа выборки
├── training/       # Обучение модели (Huber + Physics-Informed Loss)
├── weights/        # Сохраненные веса 
├── app.py          # Интерактивное веб-приложение на базе Streamlit
└── requirements.txt
```

## Быстрый старт

### 1. Установка окружения
```bash
git clone https://github.com
cd PROTEYE
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Сборка датасета и обучение
```bash
# Препроцессинг сырых PDB файлов в графы
python -m scripts.preprocess_pdb

# Генерация тренировочной выборки PROTEYE
python -m scripts.build_dataset

# Запуск тренировочного цикла на 100 эпох
python -m training.train
```

### 3. Запуск веб-интерфейса
Запуск интерактивного Streamlit-дашборда с 3D WebGL-рендерингом молекул (py3Dmol) и анализом метрик в реальном времени:
```bash
streamlit run app.py
```

## 🔮 Перспективы развития (Конформационная динамика)

Текущая архитектура PROTEYE-Денойзера является математической основой для перехода к **моделированию молекулярной динамики**:

1. **Conformational Landscape & Transitions:** Перенос весов обученного кодировщика на генеративные диффузионные модели (Diffusion-based) для предсказания путей переходов между альтернативными конформационными состояниями белка.
2. **Trajectory Generation:** Обучение авторегрессионных графовых сетей для генерации физически точных траекторий движения атомов во времени, заменяющих классические тяжелые симуляции молекулярной динамики (MD Simulation).
3. **B-Factor Prediction:** Интеграция предиктора локальной гибкости атомов (B-факторов) для автоматической оценки лабильности участков полипептидной цепи.
